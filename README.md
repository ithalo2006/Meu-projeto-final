# Agenda de Consultas em C


## Autores
Íthalo Santos Cavalcante e Júlio César Ximenes


## Descrição
Projeto desenvolvido em linguagem C que implementa uma agenda simples de consultas médicas.
O sistema permite cadastrar pacientes e agendar consultas utilizando estruturas de dados,
modularização e boas práticas de programação.

Projeto refatorado para aplicar conceitos de:
- Structs
- Modularização
- Boas práticas de código
- Organização profissional de projeto em C


## Funcionalidades
- Cadastro de pacientes (nome e idade)
- Listagem de pacientes
- Agendamento de consultas
- Listagem de consultas
- Validação de entradas
- Prevenção de consultas duplicadas


## Conteúdos de C utilizados
- Structs
- Arrays de structs
- Modularização (.c e .h)
- Include guards
- Funções auxiliares
- Compilação com múltiplos arquivos
- Validação de entrada
- Boas práticas de nomenclatura
